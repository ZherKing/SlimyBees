SlimyBees 是一款 Slimefun4 插件，其灵感来源于
[Forestry](https://www.curseforge.com/minecraft/mc-mods/forestry).
它增加了许多蜜蜂物种，可以被发现、繁殖并用于生产各种材料。

每只蜜蜂的生产力也由其等位基因值决定，

它可以通过选择性繁殖来改进，为最后阶段提供一个有趣的养老游戏（。

## 特色
目前的功能包括:
* 世界生成蜂巢
* 培育蜜蜂发现新物种的能力
* 基本品种（目前主要是蜂蜜生产）
